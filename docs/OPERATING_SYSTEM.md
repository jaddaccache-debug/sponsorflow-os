# Sponsorship Operating System

## Weekly Cadence

### Monday

- Review open sponsor leads
- Confirm next actions
- Assign owner for each high-priority prospect
- Update expected sponsorship value

### Wednesday

- Send follow-ups
- Push warm introductions
- Update negotiation notes
- Escalate blocked leads

### Friday

- Review pipeline value
- Report committed vs target value
- Identify risks
- Prepare next week’s priority list

## Pipeline Status Definitions

| Status | Definition |
|---|---|
| New | Identified but not contacted |
| Contacted | First outreach sent |
| Interested | Sponsor responded positively |
| Negotiating | Package/value discussion active |
| Committed | Verbal or written agreement reached |
| Rejected | Sponsor declined |
| Dormant | No response after follow-up cadence |

## Risk Controls

| Risk | Control |
|---|---|
| Leads lost in WhatsApp | Central tracker with owner and next action |
| Weak sponsor fit | Fit score before outreach |
| Poor package clarity | Package mapper and visibility inventory |
| Missed follow-up | Weekly cadence and next-action dates |
| Sponsor dissatisfaction | Delivery proof and fulfillment tracker |
