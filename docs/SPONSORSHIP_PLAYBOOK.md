# Sponsorship Playbook

## Prospect Scoring

Score each sponsor from 1 to 10 across:

- brand fit
- audience fit
- budget potential
- decision-maker access
- urgency
- category exclusivity value
- relationship warmth

## Outreach Sequence

1. Warm introduction where possible
2. Short positioning message
3. Meeting request
4. Package discussion
5. Follow-up with tailored value
6. Commitment confirmation
7. Fulfillment tracker

## Outreach Message Template

```text
Hi [Name], I’m reaching out regarding [event/project], a cultural platform connecting [audience] through [value proposition].

I believe [Company] could be a strong fit because [specific reason].

Would you be open to a short call this week to explore a visibility or in-kind partnership?
```

## Sponsor Package Logic

A strong sponsorship package should define:

- audience reached
- visibility assets
- exclusivity, if any
- hospitality or access
- content opportunities
- delivery proof
- post-event reporting
