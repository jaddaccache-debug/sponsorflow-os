# Security Policy

SponsorFlow OS is currently a documentation, schema, and workflow repository.

## Reporting Security Issues

If you find a security issue in any implementation, template, automation, or future app code, open a private report if GitHub security advisories are enabled, or contact the maintainer directly.

## Sensitive Data Warning

Do not commit:

- sponsor personal phone numbers
- private email lists
- private commercial agreements
- bank details
- signed sponsorship contracts
- confidential event budgets
- private attendee data

Use placeholder data in examples.
